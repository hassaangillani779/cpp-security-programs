# cpp-security-programs

> C++ programs written through a security mindset — built during active study of offensive security concepts and systems programming fundamentals.

---

## About

This repository contains security-oriented C++ programs written as part of a self-directed cybersecurity learning path running parallel to university coursework. Every program here is designed to reinforce systems-level thinking — understanding how authentication works, how attackers think, and how to build defensive logic from scratch.

---

## Programs

### Brute Force Detector
Detects repeated failed login attempts and flags suspicious activity — simulating the core logic behind account lockout systems used in real authentication infrastructure.

### Login Attempt Limiter
Enforces a maximum number of login attempts before locking access — models the rate-limiting logic found in production security systems.

### Password Strength Checker
Evaluates password strength based on length, character variety, and complexity rules — teaches the criteria used by real-world authentication systems.

### Menu Driven Cyber Security Tool
A terminal-based security toolkit with a menu interface — combines multiple security utilities into a single interactive program, demonstrating modular design and function-driven architecture.

### Security Toolkit with Functions
A consolidated set of security utility functions — demonstrates clean function design, parameter passing, and modular code organization applied to security use cases.

### Function Practice (Function1, Function 2, Function 3, Function 4)
Supporting programs written during the functions milestone — covering parameters, pass by value/reference, function overloading, and scope, all applied to security-relevant scenarios.
